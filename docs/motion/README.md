# Motion (анимации)

Принцип: анимации должны быть быстрыми, предсказуемыми и уважать настройки пользователя.

## Рекомендации

- предпочтение: `opacity` / `transform` / `colors`
- layout-анимации (например, ширина сайдбара) — через CSS `transition-*`, без JS-анимаций
- reduced motion: добавляй `motion-reduce:transition-none` и/или `motion-reduce:transform-none`

Ориентиры по длительностям:

- hover/press: 120–180ms
- раскрытия/переключатели: 180–280ms
- сложные переходы: 250–350ms
